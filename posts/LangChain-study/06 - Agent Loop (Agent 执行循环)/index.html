两种模式的选择

场景 推荐模式
显示 Agent 执行步骤 stream_mode="updates"
聊天界面打字机效果 stream_mode="messages"
调试/监控工具调用 stream_mode="updates"
用户体验优化 stream_mode="messages"

stream vs invoke

方法 返回 用途
invoke() 完整结果 等待完成后一次性获取
stream() 生成器 实时获取中间步骤/token
