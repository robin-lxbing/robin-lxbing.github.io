问题2：ssh: connect to host xx.xx.xx.xx port 22: Network is unreachable

尝试解决：

  1. 检查本机网段是否正确： 若本机 IP 与服务器不在同一网段，无法直连，例如：

    • 本机：192.23.30.123/24
    • 服务器：192.23.28.124/24 → 这是两个不同网段，ping 不通是正常的。

  2. 解决方式：

    • 改为与服务器同网段的 IP，例如： 本机设为：192.23.28.123/24

    • 或添加静态路由：

  3. 确认网线/交换机端口正常、未禁用 VLAN。
